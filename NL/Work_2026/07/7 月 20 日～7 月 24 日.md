项目名称： NeuroStudio 脑电采集分析平台

项目描述：面向脑电信号采集、处理、分析及可视化应用需求，研发一套集脑电数据管理、信号预处理、特征分析、空间分布显示和三维脑电分析于一体的软件系统，支撑科研/临床脑电全流程。

- 负责 NeuroStudio 桌面端系统架构与核心功能开发，覆盖 EEG 采集、离线分析、数据管理与业务插件扩展
- 完成脑电硬件采集全链路：串口协议、帧解析、双通道合并、阻抗检测、实时波形、数据存储，并实现设备与业务隔离。
- 实现离线 脑电信号处理管线及可配置步骤编排（滤波、重采样、参考、坏导/坏段检测等處理步驟
- 与算法团队协作共建 EEG 算法库
- 基于 QCustomPlot 自研多导脑波图组件：通道分页滚动、时间窗缩放、幅值标尺、降采样绘制、事件/标注叠层等。
- 基于 QtGraphicsView、QPainter 自绘二维脑电导联布局显示：电极布局解析与 2D 头模投影，支持导联连线编辑、标准/自定义 Montage 加载，以及阻抗色值映射与交互。
- 基于 VTK 三维渲染框架，实现脑表面模型加载、三维交互显示及 EEG 溯源结果空间映射，将分析能力从二维波形扩展到三维脑空间，为空间分析与科研应用提供可视化支撑

技术栈： C++17、Qt 6、CMake、QCustomPlot、QPainter/QGraphicsView、VTK、串口通信、实时数据处理、插件架构、REST、WebSocket、Python 嵌入、MNE/NumPy、EEG/BDF



项目名称：
基因测序数据分析平台

项目描述：
面向高通量基因测序数据分析需求，研发一套基于 C++/Qt 的专业数据分析桌面平台，实现测序数据导入、实验数据管理、生物信息分析流程调度、结果可视化及报告生成。支撑科研及临床基因检测应用。

项目职责：
- 负责基因测序分析平台架构设计与核心模块开发，基于 C++/Qt 构建模块化软件框架，实现数据处理、业务逻辑、数据库及界面展示分层设计。
- 基于 Qt Widgets 开发项目管理、样本管理、分析任务配置及结果展示等功能界面，构建完整的数据分析操作流程。
- 设计基因数据管理模块，基于 MySQL 完成项目、样本、任务及分析结果的数据建模、存储与查询，实现实验数据生命周期管理。
- 构建 C++ 分析任务调度框架，实现分析流程配置、任务队列管理、状态跟踪及结果回收，支持多种检测流程扩展。
- 开发测序数据处理与结果可视化模块，实现大规模文件解析、分析结果展示、统计图表显示及自动化报告生成。
- 参与软件工程化建设，包括数据库优化、日志系统、异常处理及性能优化，提高系统稳定性和可维护性。
技术栈：
C++、Qt、CMake、MySQL
