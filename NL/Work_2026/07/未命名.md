项目名称： NeuroStudio 脑电采集分析平台

项目描述： 单进程 Qt 桌面 EEG 宿主，支持放大器实时采集、离线分析、云端协作与问卷/范式实验任务，支撑科研/临床脑电全流程。

- 从 0 到 1 搭建分层 + 插件宿主架构（Views / Interactor / Entities / Hardware / Platform），主壳稳定、业务可插拔，用 EventBus 统一数据集与 Session 生命周期
- 完成脑电硬件采集全链路：串口协议、帧解析、双通道合并、阻抗检测、实时波形、BDF 录制，并实现设备与业务隔离
- 实现离线 脑电信号处理管线（滤波、重采样、参考、坏导/坏段检测、ICA、分段/诱发等）及可配置步骤编排。
- 与算法团队协作共建 EEG 算法库：约定 C++/Python 调用契约与数据交换格式，封装嵌入式 Python 运行时（CppBridge），对接基于 MNE/NumPy/SciPy 的预处理与可视化能力
- 基于 QCustomPlot 自研多导脑波图组件：通道分页滚动、时间窗缩放、幅值标尺、降采样绘制、事件/标注叠层，支持离线回放与实时采集双模式
- 基于 Qt Graphics View / QPainter 自绘二维 Montage：电极布局解析与 2D 头模投影，支持导联连线编辑、标准/自定义 Montage 加载，以及阻抗色值映射与交互
- 基于 VTK 三维渲染框架，实现脑表面模型加载、三维交互显示及 EEG 溯源结果空间映射，将分析能力从二维波形扩展到三维脑空间，为空间分析与科研应用提供可视化支撑

技术栈： C++17、Qt 6、CMake、QCustomPlot、QPainter/QGraphicsView、VTK、串口通信、实时数据处理、插件架构、REST、WebSocket、Python 嵌入、MNE/NumPy、EEG/BDF
