项目名称： NeuroStudio 脑电采集分析平台  
项目描述： 单进程 Qt 桌面 EEG 宿主，支持放大器实时采集、离线分析、云端协作与问卷/范式实验任务,支撑科研/临床脑电全流程。
- 从 0 到 1 搭建分层 + 插件宿主架构（Views / Interactor / Entities / Hardware / Platform），主壳稳定、业务可插拔，用 EventBus 统一数据集与 Session 生命周期
- 脑电硬件采集全链路：串口协议、帧解析、双通道合并、阻抗检测、实时波形、BDF 录制及设备与业务隔离
- 离线 EEG 处理流水线（滤波、重采样、ICA、分段/诱发等）及 C++/Python（MNE）桥接，支持预处理与可视化


技术栈： C++17、Qt 6、CMake、串口通信、实时数据处理、插件架构、REST、WebSocket、Python 嵌入、EEG/BDF
